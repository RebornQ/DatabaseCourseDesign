其实本来没打算做的，临交作业的前一个星期突然遇上了[妹子UI](http://amazeui.org/)，突然就有兴趣玩一下（大雾），就随手用原生php写了一下。

## 注意事项
1. 食用前请检查`inc`目录下的`connect.php`中的数据库配置
2. 如果是用我的数据表，超级管理员的帐号是**20180101**，密码**admin**（其实为了方便调试我把所有帐号的密码我都设**admin**了🌚）
